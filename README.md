# MintMind

An AI-powered coin identification and collection management app for coin enthusiasts, built with
Kotlin Multiplatform (Compose Multiplatform) targeting Android and iOS.

## Tech Stack

- **UI**: Compose Multiplatform
- **DI**: Koin 4.x
- **Navigation**: Jetpack Navigation Compose (type-safe sealed `Screen`)
- **Networking**: Ktor 3.x (OkHttp on Android, Darwin on iOS)
- **Database**: Room 2.8.x with KSP
- **AI/Computer Vision**: Firebase AI (Android), VertexAI (iOS)
